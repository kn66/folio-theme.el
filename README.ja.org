#+title: folio-theme

Language: [[./README.org][English]] | *日本語*

紙のようなオフホワイト背景、深い青の構造色、抑えた赤の強調色を持つ
暖かいライト Emacs テーマです。

~folio-theme~ は現在、~modus-themes~ と ~ef-themes~ を土台にした
派生テーマとして構築されています。

- ~ef-themes~ は意味ベースの palette 語彙を提供する
- ~modus-themes~ はその palette を広範な package face に適用する
- ~folio~ は独自の部分 palette と semantic mapping によって、
   紙 / 青 / 赤 のアイデンティティを維持する

[[file:assets/palette.svg]]

* 謝辞

~folio-theme~ は、~modus-themes~ と ~ef-themes~ の作者である
Protesilaos Stavrou 氏の仕事の上に成り立っています。派生テーマとして
の構造、意味ベースの palette 語彙、広い face カバレッジは両プロジェクト
に依拠しています。

* Dependencies

- Emacs ~28.1+~
- ~modus-themes 5.0.0+~
- ~ef-themes 2.0.0+~

* インストール

** Git からインストール

Emacs 29 以降では、repository の最新版を直接インストールできます。

#+begin_src emacs-lisp
  (use-package folio-theme
    :vc (:url "https://github.com/kn66/folio-theme.el"
              :rev :newest)
    :config
    (load-theme 'folio t))
#+end_src

package の autoload がインストール先を ~custom-theme-load-path~ に登録するため、
追加の path 設定なしで ~M-x load-theme~ やテーマ選択 UI から ~folio~ を
見つけられます。

** ローカルで読み込む

依存関係をインストールし、~folio-theme.el~ を含むディレクトリを
~custom-theme-load-path~ に追加してからテーマを読み込みます。

#+begin_src emacs-lisp
  (defconst folio-theme-directory "/path/to/folio-theme-directory")

  (package-install 'modus-themes)
  (package-install 'ef-themes)

  (add-to-list 'custom-theme-load-path folio-theme-directory)
  (load-theme 'folio t)
#+end_src

有効なセッション中に ~folio-theme.el~ を再読み込みすることもできます。

* Design direction

- 背景: Sakura Editor の明色背景に着想を得た ~#FFFBF0~
- 構造: 深い青
- 警告や例外: 抑えた赤
- 構文色: ~bg-main~ に対しておおむね 4.6:1 から 5.2:1 の
  WCAG AA 下限寄り
- 操作中の選択面: ~region~ は暖色、completion は淡青で明確にする
- terminal 前景色: ~bg-main~ に対して 4.5:1 以上の文字用 variant
- 目標: 背景が黄色すぎる印象にならず、長時間でも使いやすいこと

* Implementation model

- ~folio-theme-palette-partial~: テーマの基礎となる名前付き色
- ~folio-theme-palette-mappings-partial~: UI、見出し、mark、modeline、
  terminal 色を対象にした Ef 風の意味色マッピング
- ~folio-theme-custom-faces~: 継承された mapping だけでは足りない箇所に限って
  追加する package face 上書き

テーマ設計は palette-first を維持し、青は構造、赤は例外、
暖かい黄は補助的な面に置きます。
構文強調には専用の ~*-syntax~ role を使い、UI の構造色を弱めずに
コード token だけを AA 下限寄りへ寄せます。
Corfu の選択行には識別しやすい淡青の面を使い、紙らしい背景を崩さずに
現在候補を見分けやすくしています。terminal ANSI 前景色も、white や
bright 系を含めて通常文字として読めるコントラストを確保します。
